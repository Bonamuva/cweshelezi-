Folder name: Cweshelezi Enterprise Website

How to use:
1. Unzip the folder and open index.html in a browser to preview.
2. To publish: create a GitHub repository and push the folder. Then connect the repo to Vercel (or GitHub Pages). 
3. The order form uses formsubmit.co to forward orders to Cweshelezienterprice@gmail.com (one-time verification required).
